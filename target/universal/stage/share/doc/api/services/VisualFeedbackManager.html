class VisualFeedbackManager extends AnyRef

VisualFeedbackManager = the coordinator ("director").

Owns: - highlight computation/drawing and state sets - clearing highlights - canceling selections (UI/interaction state) - hand show/hide/redraw visuals

Does NOT: - move units, attack, summon, spend mana, remove units (gameplay logic)
